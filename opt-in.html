SMS Consent & Opt-In

How Kaku collects explicit consent before sending SMS text messages.

Consent for SMS Text Messages

Kaku is an AI-powered group event coordination service. Kaku sends SMS text messages to coordinate group events such as dinners, brunches, hiking trips, and other social plans. Before any coordination messages are sent, Kaku collects explicit, separate consent for SMS text messaging directly from each recipient.

Consent for SMS text messaging is collected independently and is not bundled with our Privacy Policy, Terms of Service, or any other agreement. Recipients must actively opt in by replying YES to an introductory text message before receiving any further SMS messages from Kaku.

How Consent Is Collected (Opt-In Flow)

Kaku uses a text-based (VIA_TEXT) opt-in method. Here is the exact flow:

1

Introductory SMS is sent

When a Kaku app user adds a friend to an event plan, Kaku sends a single introductory SMS text message to that friend. This message clearly identifies who added them, describes the event, and explicitly requests consent to receive SMS text messages.

2

Recipient replies YES to consent

The recipient must actively reply YES to agree to receive SMS text messages from Kaku. No further text messages are sent until the recipient explicitly opts in by replying YES.

3

Recipient can opt out anytime

Recipients can reply STOP at any time to immediately stop all SMS text messages from Kaku. Reply HELP for support information.

Exact Consent Language (Introductory SMS)

Below is the exact introductory SMS text message sent to collect consent. The consent request is prominently displayed as the primary content of the message — it is not hidden, buried in fine print, or combined with other agreements:

"Hi! Your friend [Name] is using Kaku to plan [event description] and added you. By replying YES, you agree to receive SMS text messages from Kaku for event coordination. Msg frequency varies. Msg & data rates may apply. Reply STOP to opt out. Reply HELP for help."

Key elements of this consent message:

  • Explicitly mentions SMS text messages — the phrase "SMS text messages" is used directly
  • Clearly requests consent — "By replying YES, you agree to receive SMS text messages from Kaku"
  • Identifies the sender — Kaku is named as the message sender
  • Describes message purpose — "for event coordination"
  • States message frequency — "Msg frequency varies"
  • Discloses rates — "Msg & data rates may apply"
  • Provides opt-out instructions — "Reply STOP to opt out"
  • Provides help instructions — "Reply HELP for help"

Types of SMS Text Messages Sent

After a recipient opts in by replying YES, Kaku sends the following types of SMS text messages for event coordination only:

  • Availability requests — asking if dates/times work for the recipient
  • Event confirmations — confirming finalized event details (date, time, location)
  • Event updates — notifying of changes to plans (reschedules, cancellations, location changes)
  • Follow-up coordination — additional questions about preferences, RSVPs, or logistics

Kaku does not send marketing, promotional, or advertising messages. All SMS text messages are strictly for coordinating the specific group event the recipient was invited to.

Message Frequency

Message frequency varies based on event activity. A typical event involves 2–10 SMS text messages total. Messages are only sent when there is active coordination for an event the recipient has opted into.

Opt-Out Instructions

Recipients can opt out of SMS text messages at any time:

  • Reply STOP to any message to immediately unsubscribe from all Kaku SMS text messages
  • Reply HELP to receive support contact information
  • Contact support@kaku-ai.com for assistance

Consent Is Not a Condition of Purchase

Consenting to receive SMS text messages from Kaku is not a condition of any purchase or use of the Kaku service. The Kaku app user (the event planner) can still coordinate events even if invited friends choose not to opt in to SMS messaging.
